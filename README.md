# rails-basic-blog

The Most Basic Blog app, made with Ruby on Rails 7.

## Getting Started

```sh
git clone https://github.com/alvinometric/rails-basic-blog.git
cd rails-basic-blog
```

### Install dependencies

```sh
bundle install
```

### Setup DB

(SQLite, no additional dependencies required.)

Run the migration

```sh
rails db:migrate
```

Seed the DB

```sh
rails db:seed
```

### Run the server

```sh
rails server
```
